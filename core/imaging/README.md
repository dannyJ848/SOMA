## Imaging Modality Overview

| Modality | Radiation | Contrast | Best For |
|----------|-----------|----------|----------|
| X-Ray | Low | Rarely | Bones, lungs, foreign bodies |
| CT | Higher | Often | Detailed cross-sections, emergencies |
| MRI | None | Sometimes | Soft tissue, brain, spine, joints |
| Ultrasound | None | Never | Real-time, pregnancy, vessels |
| Nuclear | Variable | Radiotracer | Functional imaging, cancer staging |

## Quick Reference: Which Study to Order

### Emergency Presentations
- **Trauma**: CT (head, C-spine, chest/abd/pelvis)
- **Stroke**: CT head (immediate), MRI (definitive)
- **PE**: CT pulmonary angiography
- **Aortic dissection**: CT angiography
- **Appendicitis**: CT abdomen/pelvis

### Common Clinical Scenarios
- **Headache**: Start MRI brain
- **Back pain with red flags**: MRI spine
- **Abdominal pain**: CT or ultrasound based on location
- **Joint pain**: X-ray first, MRI if needed
- **Chest pain**: CXR, then CT or echo based on concern
